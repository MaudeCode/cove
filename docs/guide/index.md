# Introduction

Cove is a beautiful, feature-rich WebUI for [OpenClaw](https://github.com/openclaw/openclaw) — the open-source AI assistant gateway.

## Features

- **Multi-session chat** — Manage multiple conversations with different agents and models
- **12+ themes** — Light, dark, and colorful options
- **Canvas** — Agents can push images and content directly to your browser
- **Command palette** — Quick access to everything with `Cmd+K`
- **Mobile-friendly** — Works great on phones and tablets
- **Keyboard shortcuts** — Navigate efficiently without a mouse

## Quick Start

The fastest way to get started:

```bash
npx @maudecode/cove
```

This starts Cove on `http://localhost:8080`. Connect it to your OpenClaw gateway and start chatting.
